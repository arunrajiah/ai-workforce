# Ship an AI SDR
> Replaces: SDR/BDR headcount, Apollo, Outreach, Salesforce, Calendly. Ship in: an afternoon.

This guide wires up an open-source sales stack: a CRM to hold the pipeline, a scraper to research leads, an automation engine to run outreach sequences, and a booking page for demos. An LLM sits on top to draft personalized outreach and qualify replies.

> **Try it with zero API keys.** The bundled flagship app `apps/ai-sdr` runs the whole loop out of the box — spin it up first to see the workflow before self-hosting the full stack.

## What you'll build
- [Twenty](https://github.com/twentyhq/twenty) — open-source CRM for pipeline and contacts.
- [Firecrawl](https://github.com/mendableai/firecrawl) — scrape and structure lead/company data for research.
- [n8n](https://github.com/n8n-io/n8n) — automate outreach sequences and reply handling.
- [Cal.com](https://github.com/calcom/cal.com) — self-hosted demo booking.

## Prerequisites
An LLM key and Docker. See [`/quickstart`](/quickstart) for the shared foundation.

## Step 1 — Stand up the CRM (Twenty)
```bash
git clone https://github.com/twentyhq/twenty.git
cd twenty
cp packages/twenty-docker/.env.example .env   # set APP_SECRET, PG_DATABASE_PASSWORD
docker compose up -d
```
Twenty serves on port `3000` by default. Set `SERVER_URL` in `.env` to your public host.

## Step 2 — Add lead research (Firecrawl)
```bash
git clone https://github.com/mendableai/firecrawl.git
cd firecrawl
cp apps/api/.env.example .env   # set REDIS_URL, ports
docker compose up -d
```
The API listens on port `3002`. Use `/scrape` and `/search` to enrich a lead before outreach.

## Step 3 — Add outreach automation (n8n)
```bash
docker run -d --name n8n -p 5678:5678 \
  -v n8n_data:/home/node/.n8n \
  docker.n8n.io/n8nio/n8n
```
n8n serves on port `5678`. Build a workflow: trigger on new Twenty lead → call Firecrawl → draft email → send.

## Step 4 — Add demo booking (Cal.com)
```bash
git clone https://github.com/calcom/cal.com.git
cd cal.com
cp .env.example .env   # set NEXTAUTH_SECRET, CALENDSO_ENCRYPTION_KEY, DATABASE_URL
docker compose up -d
```
Cal.com serves on port `3000` — remap if it collides with Twenty. Drop your booking link into outreach emails.

## Step 5 — Wire in the AI
In your n8n outreach workflow, add an HTTP/AI node that calls any OpenAI-compatible LLM endpoint (`LLM_BASE_URL` / `LLM_MODEL`, plus `LLM_API_KEY` for hosted gateways) to draft personalized first-touch emails from the Firecrawl research. For a fully local setup, point the node at [Ollama](https://github.com/ollama/ollama) (`http://localhost:11434/v1`). Add a second node to classify inbound replies (interested / not / needs-info) and route to the Cal.com booking link.

## Ship it
Put [Caddy](https://caddyserver.com) in front of Twenty, n8n, and Cal.com for automatic HTTPS. See [`/quickstart`](/quickstart) for the reverse-proxy snippet.
