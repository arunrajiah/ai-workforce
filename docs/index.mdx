# AI Workforce Docs

Welcome. This is the tutorial hub for **[AI Workforce](https://github.com/arunrajiah/ai-workforce)** —
the curated directory of open-source AI apps that run entire departments.

Every guide here is copy-pasteable and gets you to a **working deployment**, not a
demo. Guides render on GitHub and publish to [hub.arunrajiah.com](https://hub.arunrajiah.com).

## Start here

- **[Quickstart](/quickstart)** — the shared foundation every AI-native app needs
  (an LLM endpoint, a vector store, Docker, a domain, secrets) — set up once, reuse everywhere.
- **Pick a department** below and follow its deploy guide.

## Departments

Most departments below have a dedicated AI-native deploy guide. A few (Sales,
Support, HR, Finance, Operations) don't yet have a strong single-purpose
AI-native tool with its own guide, so [Veska](/apps/veska/deploy) — an
AI-native ERP — is the deploy guide listed there; see each department's page
in the repo for the individual AI-native tools too.

| Department | Deploy guide | You'll ship |
|---|---|---|
| Sales & Revenue | [Veska](/apps/veska/deploy) | CRM + an AI SDR that drafts outreach |
| Marketing & Growth | [Dify](/departments/marketing/deploy) | A content + SEO agent platform |
| Customer Support | [Veska](/apps/veska/deploy) | A support desk + AI ticket deflection |
| Human Resources | [Veska](/apps/veska/deploy) | HR ops + AI resume screening |
| Finance & Accounting | [Veska](/apps/veska/deploy) | Ledger, invoicing + AI receipt extraction |
| Legal & Compliance | [AnythingLLM](/departments/legal/deploy) | A private contract-review assistant |
| Engineering / DevOps | [OpenHands](/departments/engineering/deploy) | An autonomous coding agent |
| Data & Analytics | [AI Analyst](/departments/data/deploy) | A talk-to-your-data agent |
| Design & Creative | [ComfyUI](/departments/design/deploy) | A brand-asset generation pipeline |
| Operations & PM | [Veska](/apps/veska/deploy) | An AI workflow engine |
| Product & Research | [GPT Researcher](/departments/product/deploy) | A cited market-research agent |
| Executive / Assistant | [Khoj](/departments/executive/deploy) | A personal AI second brain |

## The flagship apps

Prefer to run something built-to-fit? The repo ships original reference apps in
[`apps/`](https://github.com/arunrajiah/ai-workforce/tree/main/apps), each with an
offline demo mode so you can try it with zero API keys.
