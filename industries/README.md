# 🏭 Industries — AI-native open source by vertical

The [departments](../departments) are the primary map. This index re-slices the
world by **industry**, pointing to the best **AI-native** open-source projects
(an LLM, AI agent, or ML model at the core) for each one — **52 tools across 12
verticals**.

For general back-office ops (CRM, support, finance, HR) in *any* vertical, one
platform covers it: **[Veska](../apps/veska/)** — so each page links it once and
focuses on the AI tools unique to that industry.

> **Honest note on sparsity:** some verticals (real estate, hospitality,
> insurance, logistics, e-commerce) have little *dedicated* AI-native OSS yet —
> the strongest options are often horizontal AI tools (voice agents, document
> extraction, forecasting, RAG) applied to the vertical. Each page says so plainly
> rather than padding the list.
