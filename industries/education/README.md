# 🎓 Education / EdTech — AI-native open source

> The strongest education AI is a tutor or study assistant grounded in your own
> material — an agent that answers, quizzes, and explains over course content
> via RAG. Self-host these to keep student data on your own infra and off a
> vendor's servers. (LMS platforms are excluded — they're infrastructure, not
> AI-native.)

### [DeepTutor](https://github.com/HKUDS/DeepTutor)

> An agent-native personalized tutor — chat, quizzes, visualization, and mastery practice over your materials in one workspace, replacing a per-seat AI-tutoring subscription.

| | |
|---|---|
| **Stars** | ~25k |
| **AI** | LLM agent loop + RAG across modes (chat/quiz/solve) |
| **Replaces** | paid AI-tutor SaaS (Khanmigo-style) |
| **Self-host** | Medium — configure LLM + embedding providers |
| **License** | Apache-2.0 |

---

### [Khoj](https://github.com/khoj-ai/khoj)

> An AI "second brain" over your notes and documents — a study assistant that answers, researches, and schedules, replacing a paid personal-AI/research tool.

| | |
|---|---|
| **Stars** | ~35k |
| **AI** | RAG + custom agents over your docs, any local/online LLM |
| **Replaces** | paid study-assistant / research copilot |
| **Self-host** | Easy — Docker; runs fully local with Ollama |
| **License** | AGPL-3.0 |

---

### [RAGFlow](https://github.com/infiniflow/ragflow)

> A RAG engine with deep document understanding — build a grounded, cited Q&A tutor over textbooks, lectures, and course PDFs, replacing a bespoke RAG build.

| | |
|---|---|
| **Stars** | ~84k |
| **AI** | RAG + agent capabilities with layout-aware chunking |
| **Replaces** | hand-built RAG over course material |
| **Self-host** | Medium — Docker Compose stack |
| **License** | Apache-2.0 |

---

### [Quivr](https://github.com/QuivrHQ/quivr)

> An opinionated RAG framework for chat-with-your-docs — a quick path to a curriculum/notes Q&A assistant, replacing a paid document-chat tool.

| | |
|---|---|

### [Reor](https://github.com/reorproject/reor)

> A private, local AI note-taking app that auto-links notes and answers questions over them — a self-contained study workspace, replacing a cloud note-AI subscription.

| | |
|---|---|
| **Stars** | ~8.6k |
| **AI** | Local LLM + semantic search over your notes (via Ollama) |
| **Replaces** | paid AI note-taking apps |
| **Self-host** | Easy — desktop app, fully local |
| **License** | AGPL-3.0 |
