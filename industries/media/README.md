# 🎬 Media / Creative — AI-native open source

> Media is the vertical where open-source AI is richest: image and video
> generation, speech-to-text, and voice synthesis all have mature,
> self-hostable models at their core. These replace whole categories of
> subscription creative SaaS — and keep your assets on your own hardware.

### [ComfyUI](https://github.com/Comfy-Org/ComfyUI)

> A node-based engine for diffusion workflows — image, video, and audio generation — replacing a stack of paid generative-media subscriptions.

| | |
|---|---|
| **Stars** | ~119k |
| **AI** | Diffusion models (SDXL, Flux, video models) via a visual graph |
| **Replaces** | Midjourney / paid generative-media tools |
| **Self-host** | Medium — GPU recommended, model downloads |
| **License** | GPL-3.0 |

---

### [Whisper](https://github.com/openai/whisper)

> A general-purpose speech-recognition model — transcription, subtitles, and translation — replacing a paid transcription/captioning service.

| | |
|---|---|
| **Stars** | ~104k |
| **AI** | Transformer ASR model, multilingual + translation |
| **Replaces** | paid transcription / captioning SaaS |
| **Self-host** | Easy — runs locally; faster-whisper for speed |

### [Fooocus](https://github.com/lllyasviel/Fooocus)

| | |
