# 💳 Fintech — AI-native open source

> Finance is the vertical where AI-native OSS is deepest: LLM analyst agents, multi-agent trading frameworks, purpose-built financial models, and LLM document extraction that turns statements and filings into structured data. Everything below has an LLM, agent, or trained model at its core — not a ledger with a chatbot bolted on.

### [TradingAgents](https://github.com/TauricResearch/TradingAgents)

> A multi-agent LLM framework that mirrors a trading firm — fundamental, sentiment, technical, and risk agents debate a decision — replacing a desk of junior analysts.

| | |
|---|---|
| **Stars** | ~90k |
| **AI** | Multi-agent LLM framework; specialized analyst/researcher/trader/risk agents collaborate on decisions |
| **Replaces** | junior analyst desk; discretionary research workflow |
| **Self-host** | Medium — Python, bring your own LLM + market-data keys |
| **License** | Apache-2.0 |

---

### [OpenBB](https://github.com/OpenBB-finance/OpenBB)

> The open financial-data platform for analysts, quants, and AI agents — connect data once and expose it to Python, dashboards, and AI copilots via MCP.

| | |
|---|---|
| **Stars** | ~70k |
| **AI** | Agent-first data layer; MCP servers let LLM copilots do autonomous financial research over live data |
| **Replaces** | a Bloomberg/terminal data subscription; bespoke data plumbing |
| **Self-host** | Medium — Python platform, configure data providers |
| **License** | AGPL-3.0 |

---

### [FinGPT](https://github.com/AI4Finance-Foundation/FinGPT)

> Open-source financial LLMs — cheaply fine-tuned (LoRA) for sentiment, forecasting, and financial data extraction, so you own the model instead of renting one.

| | |
|---|---|
| **Stars** | ~21k |
| **AI** | Domain-specific financial LLMs (Llama-2/Falcon/ChatGLM bases), fine-tuned for finance tasks |
| **Replaces** | paid financial-NLP APIs; closed models like BloombergGPT |
| **Self-host** | Hard — model fine-tuning + inference; GPU expected |
| **License** | MIT |

---

### [FinRobot](https://github.com/AI4Finance-Foundation/FinRobot)

> An AI-agent platform for financial analysis — chains LLM reasoning over market data to generate equity research reports and stock analysis.

| | |
|---|---|
| **Stars** | ~7.4k |
| **AI** | Financial-CoT agents combining LLMs with quant analytics for report generation and forecasting |
| **Replaces** | equity-research analyst hours; manual report drafting |
| **Self-host** | Medium — Python/notebooks, bring your own LLM + data keys |
| **License** | Apache-2.0 |

---

### [Unstract](https://github.com/Zipstack/unstract)

> LLM-powered extraction that turns statements, invoices, and filings into structured JSON from a plain-language schema — no template regex to maintain.

| | |
|---|---|
| **Stars** | ~6.7k |
| **AI** | Prompt-defined extraction over any LLM (OpenAI/Bedrock/Gemini/Ollama); deploys as an API or ETL step |
| **Replaces** | manual data entry; per-format document parsers; SaaS IDP |
| **Self-host** | Medium — Docker Compose, wire in an LLM |
| **License** | AGPL-3.0 |
